DBMS

database management system is a software that helps us to store, organize, manage, and retrieve data easily.
it is like a digital cupboard where data is stored in tabular form.

MY SQL IS AN EXAMPLE OF DBMS

DDL (Data definition language) :

to change the structure of data:

1.	create - to create tables, database etc.
2.	alter - to change the existing code
3.	drop - remove the database
4.	truncate - remove the table

DML (data manipulation language):

to change the data:

1.	select - used to retrieve data from table in a database
2.	update - used to modify existing records in a table
3.	insert - used to add new rows to a database
4.	delete - used to delete records (rows) from a database table.
